## Getting Started

The Project uses ReactJS and TailwindCSS

### Prerequisites

- Node.js (v14 or later)
- Yarn (recommended) or npm

### Installation

Clone the repository and install the dependencies:

### Using Yarn

```bash
git clone <repository-url>
cd <project-directory>
yarn install
```

### Using Npm

```bash
git clone <repository-url>
cd <project-directory>
npm install
```
